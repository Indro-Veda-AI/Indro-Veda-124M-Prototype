# 🔱 Indro-Veda (Sovereign AI)

**Indro-Veda** is a decentralized, sovereign AI initiative aimed at building high-performance Large Language Models (LLMs) specifically optimized for logic, reasoning, and Indian languages.

## 🚀 Current Phase: Prototype 1 (124M)
We are currently training a **124M parameter base model** to test our custom architecture and data pipeline before scaling to 200B+ tokens.

### 🧠 Model Specifications
- **Architecture:** Custom IndroVedaSovereign (Transformer-based)
- **Parameters:** 124 Million
- **Tokenizer:** GPT-2 (Optimized)
- **Training Hardware:** 2x Tesla T4 GPUs (Distributed Training)

## 🇮🇳 The Sovereign Vision
Our goal is to move from this 124M prototype to a **200 Billion token** foundation model that represents the digital sovereignty of India.
